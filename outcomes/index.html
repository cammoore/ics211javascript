Learning Outcomes

1. Understand Javascript

To learn about the Javascript programming language including advanced object-oriented programming techniques such as polymorphism, inheritance, and encapsulation.

The student will:

  • Design, develop and test Javascript programs using polymorphism, inheritance and encapsulation.

Referencing modules: OOP and Class Hierarchies, Big-O Notation, Lists, Midterm Exam 1

Assessed by: Object Oriented Programming, ArrayList and Simple Sorting, Double Linked List with Simple Sorting, List Iterators, ArrayList, ArrayList, LinkedList, LinkedList, OOP, Arrays, Lists, Big-O and Quadratic Sorting

2. Object Oriented Programming in Javascript

To learn advanced object-oriented programming techniques in Javascript.

The student will:

  • Use polymorphism, inheritance, and encapsulation in substantial Javascript programs, when appropriate reusing and extending previously created classes.

Referencing modules: OOP and Class Hierarchies

Assessed by: Object Oriented Programming, ArrayList and Simple Sorting, Double Linked List with Simple Sorting, List Iterators, ArrayList, ArrayList, LinkedList, LinkedList

3. Program in Javascript

To learn how to produce robust programs in Javascript using exception handling and extensive program testing.

The student will:

  • Develop substantial Javascript programs, when appropriate reusing previously created classes, writing programs requiring three or more classes.

Referencing modules: Welcome to ICS 211, OOP and Class Hierarchies, Lists, Midterm Exam 1

Assessed by: Object Oriented Programming, ArrayList and Simple Sorting, Double Linked List with Simple Sorting, List Iterators, ArrayList, ArrayList, LinkedList, LinkedList, OOP, Arrays, Lists, Big-O and Quadratic Sorting

4. Software Engineering

To be exposed to software engineering techniques such as: abstract data types, separation of definition and implementation, encapsulation, information hiding, code reviews, and testing.

The student will:

  • Develop abstract data types.

  • Conduct personal and peer code reviews.

  • Choose, design and develop appropriate tests for their Java code.

Referencing modules: OOP and Class Hierarchies

8. Big O Notation

To learn and uses the basics of algorithm analysis, including big-O notation.

The student will:

  • Calculate and explain the big-O runtime of Java code.

Referencing modules: Big-O Notation, Midterm Exam 1

Assessed by: ArrayList and Simple Sorting, Double Linked List with Simple Sorting, ArrayList, ArrayList, LinkedList, LinkedList, OOP, Arrays, Lists, Big-O and Quadratic Sorting

10. Lists

To learn and implement the List standard data structure. Know the standard interface for a List.

The student will:

  • Implement two different versions of the List interface.

  • Explain the trade-offs between the two different implementations.

Referencing modules: Lists, Midterm Exam 1

Assessed by: ArrayList and Simple Sorting, Double Linked List with Simple Sorting, List Iterators, ArrayList, ArrayList, LinkedList, LinkedList, OOP, Arrays, Lists, Big-O and Quadratic Sorting
