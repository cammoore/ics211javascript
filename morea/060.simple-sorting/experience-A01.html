ICS 211 Homework A01: Sorting Arrays

  • Create a Generic Javascript class named ArraySort that has three functions:
public void insertionSort(int[] data);
public void bubbleSort(int[] data);
public void selectionSort(int[] data);

Each method should implement the named simple sort algorithm on the array data. The ‘smallest’ element should be in position 0, ‘largest’ in array.length - 1.

The methods should keep track of the number of comparisons, number of swaps and the amount of time it took to sort the array. To record the time taken, your code should call getTime() at the beginning of each of the methods and save the value as the start time. It should then call getTime() again at the end, and print the difference between the two times. The difference may be zero if your system clock does not update the time quickly enough, but otherwise should accurately report how many milliseconds it took to execute your code. When the sort methods complete they should report the number of comparison, swaps, and the execution time in nanoseconds.

Be sure to thoroughly test your code. Turn in your test code along with your Class.

Algorithm Analysis

Write up a runtime analysis of each of the sorting methods, giving the big-O of each, and explaining how you got your results. Send your analysis to the TA together with your code. This part counts for 25% of the grade on this assignment. Even if your code doesn’t work, you should do this part to the best of your ability.
